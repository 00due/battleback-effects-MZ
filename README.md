# battleback-effects-MZ
 Battle background effects for RPG Maker MZ

This plugin allows you to create battle background effects similar to those in Earthbound and other games.

## Download instructions
You may want to choose the "classic" version of the plugin if you need maximum compatibility. The newest version of the plugin has made major changes to the way battle backgrounds work. The plugin adds a new multi-layer battle background system, which allows you to add more than just 2 backgrounds. However, this means that the older system is no longer compatible with this plugin. If you heavily rely on the default way of handling battle backgrounds, please download the "classic version" of this plugin!

Effects included:
```
- wave
- twist
- scroll
- hue / brightness shift
- blend modes (not included in classic version)
- infinite amount of background layers (not included in classic version)
```
More effects may be added in future releases. All the effects can be customized.
